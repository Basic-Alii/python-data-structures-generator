# Python Data Structures Generator 🐍

An interactive Python command-line utility that demonstrates input parsing and conversion into core Python built-in data collections: **Lists**, **Tuples**, and **Sets**.

---

## ✨ Features

- **Dynamic Data Creation**: Converts comma-separated input strings into native Python data structures on the fly.
- **Data Structure Demonstrations**:
  - **List**: Preserves order and allows duplicate values.
  - **Tuple**: Demonstrates immutable sequence creation.
  - **Set**: Automatically eliminates duplicate entries to show unique collection behavior.
- **Interactive Menu**: Runs continuously until the user chooses to exit.
